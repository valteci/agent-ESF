# Objetivo do projeto

Estamos criando um agente que vai pegar dados do usuário e gerar uma saída json correspondente.
Para isso estamos criando SKILLs para esse agente.

Você é responsável por avaliar as SKILLs que existem dentro do arquivo .agents para gerar novas SKILLs conforme explicado pelo usuário. O arquivo de AGENTS.md desse agente está na raiz do projeto com o nome de "__AGENTS.md"
